大学の文化祭で展示する作品として個人で開発しました。

大量の広告に埋め尽くされた画面から、「✕」ボタンを探し出して消していく、Webサイトの閲覧体験を意識したタイムアタックゲームです。

文化祭ということもあり、学生個人や屋台を出店しているサークルに広告の募集をかけたところ、多くの方々に広告を作成していただけました。ありがたい限りです。
